>#### 기본용어
><식별자>

&nbsp;&nbsp;생성자 함수의 이름은 항상 대문자로 시작 ex)will out -> willOut
&nbsp;&nbsp;변수, 함수, 속성, 메소드의 이름은 항상 소문자로 시작 ex) will return -> willReturn
